- `ALL-GREEN` — если действий уровня человека (отправить, встретиться,
  подписать, ответить на гейт) сегодня нет;
- `ACTION: <самое срочное действие одной фразой ≤120 символов>` — если есть.

Дальше (только если ACTION): ≤5 пунктов, каждый — действие + почему сегодня.
Простой русский, без жаргона системы (никаких atom/verify/гейт-номеров без
расшифровки).
